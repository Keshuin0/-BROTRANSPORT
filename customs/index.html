Customs Compliance

Frictionless transborder transit. Our automated compliance engine integrates directly with CBSA and CBP, verifying ACE and ACI manifests before your shipments reach the border.

COMPLIANCE_PIPELINE // U.S. IMPORTATION

PAPS / ACE eManifest

For shipments moving from Canada into the United States, the system coordinates the **Pre-Arrival Processing System (PAPS)**. Each trailer load is assigned a unique PAPS number using our Standard Carrier Alpha Code (**SCAC: `BROP`**).

Regulatory Agency: U.S. Customs & Border Protection (CBP)
Identifier Scheme: SCAC + SCN (Shipment Control Number)
Submission Threshold: Minimum 1 hour prior to port arrival
Electronic Interface: Automated Broker Interface (ABI) to ACE
COMPLIANCE_PIPELINE // CANADIAN IMPORTATION

PARS / ACI eManifest

For northbound transits entering Canada, our operations coordinate the **Pre-Arrival Review System (PARS)** and Advance Commercial Information (**ACI**) eManifest. Manifests are transmitted at least 1 hour before arrival.

Regulatory Agency: Canada Border Services Agency (CBSA)
Identifier Scheme: Carrier Code + Cargo Control Number (CCN)
Submission Threshold: Minimum 1 hour prior to port arrival
Release Verification: Release Notification System (RNS) sync
